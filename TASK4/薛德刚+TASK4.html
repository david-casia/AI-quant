一、海龟策略核心思想与关键优势

海龟策略（Turtle Trading Strategy）是量化交易史上最著名的趋势跟踪策略之一，起源于1980年代美国交易员理查德·丹尼斯（Richard Dennis）与威廉·埃克哈特（William Eckhardt）进行的一场传奇实验。丹尼斯相信交易能力可以通过系统化的规则传授，而非依赖天赋直觉。他从数千名申请者中招募了一群背景各异的"海龟"学员，向他们传授了一套完整的机械化交易系统，并在随后数年中获得了超过年均80%的惊人收益率，从而证明了系统化趋势跟踪策略的有效性。

海龟策略的核心思想可以概括为"通道突破+趋势跟踪+严格风控"。策略不预测市场走势，而是在价格突破历史高低点形成的通道时顺势入场，通过价格突破的动量效应捕捉趋势行情。当价格创出N日新高时买入，创出M日新低时卖出，以"截断亏损、让利润奔跑"为基本理念。这种设计使得海龟策略天然适合捕捉大级别趋势行情，而在震荡市中则会产生一定数量的虚假信号。

海龟策略的关键优势体现在以下几个方面。第一，规则清晰明确。海龟策略的所有交易决策均由预先定义的规则自动触发，不依赖主观判断，有效消除了情绪干扰和人为失误，使策略具有高度的可复制性和可回测性。第二，风险控制严密。策略引入了平均真实波幅（ATR）作为动态止损工具，根据市场波动率自动调整止损距离，在波动加剧时放宽止损以避免被噪声洗出，在波动减小时收紧止损以保护利润。第三，趋势跟踪能力强。在明确的趋势行情中，通道突破信号能够准确捕捉趋势的起始点，并在趋势结束前持有仓位，最大化趋势利润。第四，参数自适应性强。ATR的使用使策略能够自动适应不同波动率水平的市场环境，无需针对不同标的频繁调参。第五，可扩展性好。海龟策略框架可应用于股票、期货、外汇等多种资产类别，且支持多周期、多标的组合配置，分散单一品种的风险。

需要指出的是，海龟策略并非完美无缺。其最大的局限性在于震荡市中表现不佳——当价格在通道内反复波动但无法形成有效突破时，策略会产生连续的虚假突破信号，导致多次小额止损。此外，海龟策略的信号滞后性也是一个固有问题：突破信号出现在趋势已经启动之后，必然牺牲一部分趋势初期的利润。因此，在实际应用中，通常需要结合趋势过滤指标（如ADX、移动平均线斜率等）来过滤震荡市的虚假信号，提升策略的整体表现。

二、唐奇安通道、ATR与止损条件

海龟策略的技术基础由三个核心组件构成：唐奇安通道、平均真实波幅（ATR）和基于ATR的止损条件。这三个组件分别负责信号生成、波动率度量和风险控制，共同构成了海龟策略的完整技术框架。

唐奇安通道（Donchian Channel）是由技术分析先驱理查德·唐奇安（Richard Donchian）发明的价格通道指标，是海龟策略信号生成的核心工具。通道由三条线构成：上轨为过去N个交易日的最高价，下轨为过去N个交易日的最低价，中轨为上下轨的均值。在海龟策略中，通常设置两个不同周期的通道：入场通道和出场通道。入场通道周期较长（如20日），用于捕捉中长期趋势的突破信号；出场通道周期较短（如10日），用于在趋势反转时及时退出。具体而言，当当日收盘价突破入场通道上轨（即创20日新高）时，产生买入信号；当当日收盘价跌破出场通道下轨（即创10日新低）时，产生卖出信号。值得注意的是，海龟法则使用"前N日"的数据计算通道（即shift(1)后取rolling），不包含当日数据，以避免未来数据泄露。唐奇安通道的优点在于计算简单、信号明确、不依赖参数拟合，能够客观反映价格波动的极端边界。

平均真实波幅（Average True Range，简称ATR）是由技术分析大师威尔斯·威尔德（Welles Wilder）发明的波动率指标，用于衡量市场价格的波动幅度。ATR的计算分为两步：首先计算每日的真实波幅（True Range, TR），TR为以下三个值中的最大值：当日最高价与最低价之差（high - low）、当日最高价与前日收盘价之差的绝对值（|high - prev_close|）、当日最低价与前日收盘价之差的绝对值（|low - prev_close|）。后两个分量考虑了跳空缺口的影响，使得TR能够更全面地反映价格的真实波动范围。然后在TR的基础上计算N日（通常为20日）的指数移动平均，即得到ATR。海龟策略使用Wilder平滑法（alpha=1/N）进行EMA计算，这与标准的EMA略有不同，但更符合原始海龟法则的定义。ATR的值越大，说明市场波动越剧烈；ATR的值越小，说明市场波动越平缓。ATR的关键特性在于它是一个动态指标，能够随市场波动率的变化而自动调整。

止损条件是海龟策略风险管理的核心机制。海龟法则使用基于ATR的动态止损策略，其核心规则为：买入后立即设置止损价为（买入价 - 止损倍数 × ATR），其中止损倍数通常取2.0。这意味着，当价格从买入价下跌超过2倍ATR时，触发止损卖出。ATR止损的精妙之处在于其自适应性：在波动率高的市场环境中，ATR值较大，止损距离相应放宽，避免因正常波动而被"洗出"；在波动率低的市场环境中，ATR值较小，止损距离收紧，及时控制损失。此外，海龟策略还采用跟踪止损（Trailing Stop）机制：持仓期间，随着价格上涨，止损价也会相应上移（仅上移不下移），从而锁定已获利润。止损价的更新公式为：新止损价 = 当日收盘价 - 止损倍数 × ATR。当新计算出的止损价高于当前止损价时，更新止损价；否则保持不变。这一机制确保了策略在趋势延续时能够保护利润，在趋势反转时能够及时退出。

以上三个组件的协同工作机制如下：唐奇安通道负责捕捉趋势突破信号，ATR负责度量当前市场的波动率水平，ATR止损负责在趋势判断失误时控制单笔交易的最大亏损。三者结合，形成了一个"入场有信号、持仓有跟踪、出场有止损"的完整交易闭环，体现了海龟策略"系统化、机械化、风控优先"的核心设计理念。

# 1. 加载股价数据
df = pd.read_csv('002594_daily.csv')
df = df.sort_values('trade_date').reset_index(drop=True)

# 2. 计算唐奇安通道（20日入场 / 10日出场）
df['entry_upper'] = df['high'].shift(1).rolling(window=20, min_periods=1).max()
df['entry_lower'] = df['low'].shift(1).rolling(window=20, min_periods=1).min()
df['exit_lower']  = df['low'].shift(1).rolling(window=10, min_periods=1).min()

# 3. 计算ATR（20日Wilder平滑）
prev_close = df['close'].shift(1)
df['tr'] = pd.concat([
    df['high'] - df['low'],
    (df['high'] - prev_close).abs(),
    (df['low'] - prev_close).abs()
], axis=1).max(axis=1)
df['atr'] = df['tr'].ewm(alpha=1/20, min_periods=1, adjust=False).mean()

# 4. 生成交易信号（突破上轨买入，跌破下轨卖出，ATR止损）
#    止损价 = 买入价 - 2.0 × ATR，跟踪止损仅上移
for i in range(len(df)):

        signal = 1  # 买入

    elif holding == 1 and (close < stop_price or close < exit_lower):
        signal = -1  # 卖出

# 5. 回测 & 计算绩效指标
