> 版本：v1.0 | 日期：2026-07-07 | 作者：薛德刚

## 1. 产品概述

### 1.1 定位

一款纯前端（HTML/CSS/JS）的股票技术指标交互式分析工具，支持选股、调参、实时重绘四大经典指标（RSI / MACD / 布林带 / KDJ），帮助用户直观理解参数变化对指标走势的影响。

### 1.2 核心价值

| 痛点 | 解决方案 |
|------|----------|
| Python脚本改参数→重新运行→看图，迭代慢 | 滑块拖动实时重绘，毫秒级响应 |
| 一次只能看一只股票一个指标 | 多股票切换 + 四指标同屏对比 |
| 参数固定无法感知灵敏度差异 | 每个指标独立参数面板，支持预设/自定义 |

### 1.3 目标用户

- 量化交易课程学生（学习指标原理）
- 个人投资者（快速浏览多只股票指标状态）

### 2.2 四指标可调参数

```
RSI
├── period（周期）     范围 2~50    默认 14    步长 1
├── overbought（超买线） 范围 60~90  默认 70    步长 1
└── oversold（超卖线）  范围 10~40  默认 30    步长 1

MACD
├── fast（快线周期）   范围 3~30   默认 12    步长 1
├── slow（慢线周期）   范围 10~60  默认 26    步长 1
└── signal（信号线）   范围 3~30   默认 9     步长 1

布林带
├── period（周期）     范围 5~60   默认 20    步长 1
└── num_std（标准差倍数）范围 1.0~3.5 默认 2.0 步长 0.1

KDJ
├── n（RSV周期）       范围 3~30   默认 9     步长 1
├── m1（K平滑）        范围 2~10   默认 3     步长 1
└── m2（D平滑）        范围 2~10   默认 3     步长 1
```

## 3. 技术架构

### 3.1 整体架构（纯前端方案）

```
┌─────────────────────────────────────────────────┐
│                   浏览器 (Browser)                 │
│                                                   │
│  ┌──────────┐  ┌──────────────┐  ┌────────────┐  │
│  │ 数据层    │  │  计算引擎     │  │  渲染层     │  │
│  │          │  │              │  │            │  │
│  │ CSV解析   │→ │ RSI/MACD/   │→ │ Chart.js   │  │
│  │ API请求   │  │ BOLL/KDJ    │  │ 画布渲染    │  │
│  │ 数据缓存   │  │ (纯JS实现)   │  │ 交互事件    │  │
│  └──────────┘  └──────────────┘  └────────────┘  │
│                                                   │
│  数据流：选股/调参 → 计算引擎重算 → 图表重绘        │
└─────────────────────────────────────────────────┘
```

### 3.2 技术选型

| 层 | 方案 | 选型理由 |
|----|------|----------|
| 框架 | 原生HTML/CSS/JS 或 Vue3 CDN | 单文件部署，无需构建工具 |
| 图表 | **Chart.js 4.x** (CDN) | 轻量(~60KB)、免费、交互好 |
| 计算 | 纯JavaScript | 指标算法简单，无需后端 |
| 数据 | 本地CSV / Tushare REST代理 | CSV离线可用；API实时获取 |
| 部署 | 单HTML文件 / GitHub Pages | 零服务器成本 |

### 3.3 数据流设计

```
用户操作（选股/调参/改日期）
        │
        ▼
   ┌─ debounce 150ms ─┐    （防止滑块拖动时频繁重算）
   │                  │
   ▼                  ▼
 读取缓存数据      参数是否变化？
   │                  │
   ▼                  ├─ 是 → 重新计算指标
  原始OHLCV           │
   │                  └─ 否 → 直接重绘
   ▼
 指标计算引擎
   │
   ▼
 Chart.js 更新 datasets
   │
   ▼
 画面刷新（仅更新数据，不重建图表实例）
```

## 4. 界面设计

### 4.1 布局结构（行配对布局）

每个指标独占一行，左侧参数面板与右侧图表严格1:1对应：

```
┌─────────────────────────────────────────────────────────┐
│  顶栏：工具名 + 股票搜索框 + 日期范围 + 复权方式          │
├─────────────────────────────────────────────────────────┤
│  收盘价主图（全宽，K线/折线切换）                          │
├──────────────┬──────────────────────────────────────────┤
│  RSI 参数面板  │  RSI 图表                                │
│  [滑块组]      │  （拖动滑块→此图立即重绘）                 │
│  [预设/恢复]   │                                          │
├──────────────┼──────────────────────────────────────────┤
│  MACD 参数面板 │  MACD 图表                               │
│  [滑块组]      │  （独立重绘，不影响其他指标）               │
│  [预设/恢复]   │                                          │
├──────────────┼──────────────────────────────────────────┤
│  BOLL 参数面板 │  布林带图表                               │
│  [滑块组]      │                                          │
│  [预设/恢复]   │                                          │
├──────────────┼──────────────────────────────────────────┤
│  KDJ 参数面板  │  KDJ 图表                                │
│  [滑块组]      │                                          │
│  [预设/恢复]   │                                          │
├─────────────────────────────────────────────────────────┤
│  底栏：四指标状态摘要（超买/超卖/金叉/死叉）               │
└─────────────────────────────────────────────────────────┘
```

**布局核心规则**：
- `grid-template-columns: 180px 1fr`，左固定右自适应
- 每行高度由内容撑开，参数面板和图表共享同一行高
- 参数面板背景色与图表区区分（secondary vs primary）
- 行间用0.5px border分隔，视觉边界清晰

### 4.2 交互流程

```
1. 页面加载
   → 默认加载 002594.SZ 近1年前复权数据
   → 四指标用默认参数计算并渲染

2. 切换股票
   → 顶栏搜索框输入代码/名称
   → 下拉联想匹配
   → 选中后重新拉取数据 → 重算四指标 → 重绘

3. 调节参数
   → 左侧栏滑块/输入框变化
   → debounce 150ms
   → 仅重算该指标 → 仅更新该子图（不重绘其他三个）

4. 切换日期范围
   → 快选按钮 or 自定义日期
   → 从缓存截取子集 → 重算四指标 → 重绘全部

5. 鼠标悬停图表
   → 十字光标 + tooltip显示该日所有指标数值
```

### 4.3 参数面板设计规范

每个指标参数面板统一结构：

```
┌─ RSI 相对强弱指数 ──────────── [恢复默认] ─┐
│                                            │
│  周期        14                             │
│  ──●───────────────────────                │
│  2                              50         │
│                                            │
│  超买线      70          超卖线    30       │
│  ──────●────             ──●───────        │
│  60        90            10       40       │
│                                            │
│  预设： [保守 20/80/80] [标准] [激进 10/75/25] │
└────────────────────────────────────────────┘
```

- 滑块左端标注最小值，右端标注最大值
- 滑块上方实时显示当前值
- 预设按钮一键切换参数组合
- 「恢复默认」恢复到标准参数

### 4.4 状态摘要栏

底栏实时显示四指标当前状态：

```
RSI(14)  46.64  中性     │  MACD(12,26,9)  DIF<DEA  死叉    │  BOLL(20,2)  中轨下方  │  KDJ(9,3,3)  J=103  超买
```

## 8. 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| Chart.js大数据量卡顿 | 1000+数据点时交互延迟 | 降采样/Web Worker |
| Tushare API跨域 | 浏览器直接调用被CORS拦截 | 需后端代理或JSONP |
| 前复权数据需实时计算 | 前端无复权因子 | API直接请求adj=qfq |
